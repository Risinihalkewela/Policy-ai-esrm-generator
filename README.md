## 📌 Project Summary

**Policy AI** is an AI-powered system that converts complex **Environmental and Social Risk Management (ESRM)** policies into:

* Structured summaries for rapid understanding
* Scenario-specific frameworks for practical application

The system targets real-world workflows in:

* Banking & credit risk
* ESG compliance
* Sustainable finance decision-making

---

## ⚙️ Core Functionality

### 1. Policy Summarisation Engine

* Extracts:

  * Objectives
  * Risk controls
  * Governance structure
* Outputs **clean, structured summaries** suitable for decision-making

### 2. Scenario-Based Generator

* Produces **sector-specific ESRM frameworks**
* Supports contexts such as:

  * Infrastructure financing
  * Agribusiness lending
  * SME green finance
* Applies **context-aware reasoning (not template-based output)**

## 🧠 Technical Highlights

* **Backend**: Flask (REST API design, routing, request handling)

* **NLP Approach**:

  * Structured prompt design
  * Context injection (policy → scenario transformation)
  * Output formatting for usability
* **Document Processing**:

  * PDF text extraction (pdfplumber / PyPDF)
* **Frontend**:

  * Custom HTML/CSS UI
  * Interactive dual-panel workflow
